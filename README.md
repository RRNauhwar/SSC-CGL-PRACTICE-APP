# SSC-CGL-PRACTICE-APP

An ambitious, AI-powered SSC preparation platform (codename **Prayaas**) focused on **SSC CGL & CHSL**, designed to become the single destination for every SSC aspirant — combining the largest organized Previous Year Question (PYQ) database with adaptive learning, an AI mentor, deep analytics, spaced-repetition revision, realistic exam simulation, and a world-class, offline-first experience.

> Design filter for every feature: **"Will this increase the student's final SSC score?"**
